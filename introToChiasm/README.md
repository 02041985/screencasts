# What's the Point of Chiasm?

 * Component framework
   * interactive graphics
   * data loading
   * data transformation
   * data visualization
 * Application platform
   * instantiating and configuring components
   * mapping data to visualizations
   * responding to browser resize
   * linking visualizations together
