# Introduction to D3

This tutorial covers fundamental [D3.js](http://d3js.org/) concepts from the ground up, and introduces basic visualizations.

 * fundamental D3 concepts
   * selections
   * scales
   * axes
   * graphical elements
     * rectangles
     * circles
     * lines
 * visualizations
   * bar chart
   * scatter plot
   * line chart
