What is Knockout.js?

  • A "Model View Viewmodel" user interface framework
  • Core functionality
    • Data Binding - synchronize view with viewmodel
    • Computed Properties - automatic change propagation
  • Related Frameworks
    • Durandal.js A complete Single Page App framework built on Knockout, Require.js and JQuery
    • Angular.js
    • Ember.js
    • Backbone.js
    • React.js
    • Can.js
    • JSViews
    • Spine.js
